# Yardstick Gaming Benchmark

Yardstick is a benchmark for Modifiable Virtual Environments (MVEs).
Learn how to benchmark your MVE by reading the [tutorial](docs/tutorial.md)
or run an existing benchmark such as the [flying spectator](docs/flying-spectator-benchmark.md) or [chicken farm](docs/chicken-farm-workload.md).

See also our [report](docs/Lab_Report.pdf) on measuring performance evolution.
